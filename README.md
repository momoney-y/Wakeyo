# Smart Alarm Clock 🕒

A Python Tkinter-based smart alarm that forces you to solve a word puzzle to turn it off.

## Features
- Custom alarm time with live countdown
- Puzzle-based dismissal (guess the correct word)
- Increasing alarm intensity if wrong answers
- Add your own words and clues
- Modern UI with interactive controls

## How to Run
1. Install Python 3
2. Run: `build.bat`

## Notes
- Works on Windows (sound) and others (system beep)
- Uses only built-in Python libraries
